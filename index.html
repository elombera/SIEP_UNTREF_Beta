¿Qué es el DOI y qué usamos?

El sistema SIAP-UNTREF utiliza Crossref para consultar automáticamente los datos del artículo ingresando su DOI (Digital Object Identifier). Esta base internacional garantiza resultados confiables para artículos científicos registrados.

¿Qué es el ISSN y qué usamos?

El sistema SIAP-UNTREF utiliza OpenAlex para consultar datos sobre revistas científicas a partir del ISSN (International Standard Serial Number). Esto permite completar automáticamente campos como nombre de la revista y editorial.
